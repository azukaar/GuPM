# GuPM

Global Universal Project Manager -- Package manager, cli tool, scripts for all your projects and your system

## Dependency Manager
### install
```
gupm install
gupm install -p npm
```
### add
```
use default repo

gupm add mysql
gupm add node:react

use NPM

gupm add npm://react@1
gupm add -p npm react@1
```
