# install

`gupm install [--provider=]`

# global 

`gupm global [--provider=] tag:name@version`

# add 

`gupm add [--provider=] tag:name@version`

# version

`gupm version [--provider=]  [major|minor|patch]`

# remove

`gupm remove [--provider=] name`

# clean

`gupm clean [--provider=]`
